# Stock Valuation Band

Obsidian 문서 안에 주식 가치 밴드 계산기를 삽입하는 플러그인입니다.

## 사용법

리본의 계산기 아이콘을 선택하거나 명령 팔레트에서 **주식 가치 밴드 계산기 삽입**을 실행하면 현재 커서 위치에 아래와 같은 코드블럭이 삽입됩니다.

````markdown
```stock-valuation
guid: 23b625b5-0d86-45f5-a11f-10c7cddef698
```
````

Reading view에서 이 코드블럭은 인터랙티브 계산기로 렌더링됩니다. 예상 연간 순이익은 `억` 단위로, 현재 주가는 `원` 단위로 입력합니다.

- 예상 연간 순이익과 PER은 최소값과 최대값을 직접 입력합니다.
- 중간값은 1% 단위 슬라이더로 조정합니다.
- **시총 고정**을 켜면 중간 시가총액을 유지하도록 예상 연간 순이익과 PER 중간 슬라이더가 서로 반대로 보정됩니다.
- 총 주식 수를 입력하면 예상 주가가 계산됩니다.
- 현재 주가까지 입력하면 현재가 대비 예상 여력이 계산됩니다.
- **실시간 현재가 사용 (약 20분 지연)**을 켜면 현재 문서 frontmatter의 `symbol`과 `market`을 사용해 Yahoo Finance에서 현재가를 가져옵니다.

코드블럭에는 `guid`만 저장됩니다. 실제 입력 데이터는 플러그인 데이터 저장소에 `guid` 기준으로 저장됩니다. 여러 문서에서 같은 `guid`를 사용하면 같은 계산 데이터를 공유합니다.

**결과 복사**를 선택하면 계산 결과 텍스트가 클립보드에 복사됩니다.

## 시나리오

계산 결과 아래의 **시나리오 추가**를 선택하면 순이익과 PER 가정을 별도로 저장할 수 있습니다. 입력값은 현재 계산기의 중간값으로 시작하며, 저장 당시의 총 주식 수와 화면에 표시된 현재가를 사용해 적정가와 현재가 대비 예상 여력을 계산합니다.

저장된 시나리오는 이후 계산기나 실시간 현재가가 변경되어도 바뀌지 않는 스냅샷입니다. 기본적으로 최근 시나리오부터 표시되며, 표의 컬럼 제목을 선택하면 시나리오명, 가정, 적정가, 현재 대비 또는 저장 시점 기준으로 정렬할 수 있습니다. 정렬 상태는 `guid`별로 저장되어 뷰가 다시 로딩되어도 유지됩니다. 각 항목의 삭제 버튼을 두 번 선택하면 삭제됩니다. 같은 `guid`를 사용하는 계산기는 저장된 시나리오도 공유합니다.

**JSON 복사**를 선택하면 현재 화면의 정렬 순서대로 시나리오 테이블을 JSON으로 클립보드에 복사합니다. JSON에는 이 데이터가 사용자의 관찰과 가정을 바탕으로 만든 가치평가 시나리오라는 설명, 단위, 계산식, 추출 시점이 함께 포함되어 GPT 같은 도구에 바로 붙여넣어 질문하기 좋습니다. **질문 포함**을 켜면 공통 질문 템플릿을 적용한 문장과 JSON이 함께 복사됩니다.

**질문 템플릿**을 선택하면 플러그인 전체에서 공통으로 사용할 질문 템플릿을 수정할 수 있습니다. 템플릿에는 `{{asset_name}}`, `{{symbol}}`, `{{json}}`, `{{scenario_count}}`, `{{exported_at}}` 치환 문자열을 사용할 수 있으며, `{{json}}`은 반드시 포함해야 합니다. 템플릿이 `{{asset_name}}` 또는 `{{symbol}}`을 사용하지만 문서 frontmatter에 해당 값이 없으면 복사하지 않고 안내 메시지를 표시합니다.

## 실시간 현재가

실시간 현재가 기능은 현재 `KOSPI`, `KOSDAQ` 종목만 지원합니다. 문서 frontmatter에 아래 필드가 필요합니다.

```yaml
symbol: "003230"
market: KOSPI
```

`market`은 `KOSPI`, `KS`, `KOSDAQ`, `KQ` 값을 지원합니다. KOSPI 종목은 Yahoo Finance의 `.KS`, KOSDAQ 종목은 `.KQ` 심볼로 조회합니다.

이 기능을 켜면 `symbol`과 `market` 값이 Yahoo Finance 요청 URL에 포함됩니다. 조회 가격은 플러그인 데이터에 저장하지 않고, 같은 세션에서 60초 동안만 메모리 캐시로 사용합니다.

## 개발

의존성을 설치합니다.

```bash
npm install
```

개발 빌드를 실행합니다.

```bash
npm run dev
```

프로덕션 빌드를 실행합니다.

```bash
npm run build
```

린트를 실행합니다.

```bash
npm run lint
```
